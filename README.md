# Claude Code Proactive AIgent Plugin

Claude Code plugin for scheduling Telegram messages with natural language.

## Installation

```bash
# Install the CLI first
npm install -g proactive-aigent

# Install the plugin
claude plugin install claude-code-proactive-aigent
```

## Usage

Use the `/proactive-aigent` skill in Claude Code:

```
/proactive-aigent Schedule "Hello world" at 7 AM daily
/proactive-aigent List my schedules
/proactive-aigent Remove schedule #2
/proactive-aigent Test schedule #1
```

## Requirements

- Claude Code
- proactive-aigent CLI installed globally
- Configured bot token and chat ID
